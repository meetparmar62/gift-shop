# 🎁 S N Gift - Complete Admin Panel System

A professional, fully-functional admin panel for the S N Gift website with comprehensive CRUD operations and enhanced analytics dashboard.

---

## ✨ Features

### 🔐 Authentication
- Secure login system
- Session-based authentication
- Default credentials: `admin` / `admin123`

### 📊 Enhanced Dashboard Analytics
- **Product Analysis**: Average price, highest/lowest priced, total inventory value
- **Category Distribution**: Visual breakdown with progress bars
- **Review Analytics**: Average rating, 5-star reviews, top locations
- **Business Metrics**: Years experience, customers, gifts delivered, overall rating
- **Price Range Analysis**: Visual bar charts showing product distribution

### 🛍️ Management Systems

#### 1. Products Management
- Add/Edit/Delete products
- Category assignment
- Price management
- Badge system (Bestseller, New, Sale)
- Image URL management

#### 2. Categories Management
- Create gift categories
- Custom Font Awesome icons
- Description editing

#### 3. Features Management ("Why Choose Us")
- Add business features
- Custom icons
- Edit descriptions

#### 4. Reviews Management
- Customer review CRUD
- 5-star rating system
- Location tracking
- Review text management

#### 5. Instagram Gallery
- Add/remove images
- Grid layout management
- Profile linking

#### 6. Contact Information
- Store address editing
- Phone numbers
- Business hours
- Social media handles
- Google Maps integration

#### 7. Statistics Tracking
- Update business metrics
- Real-time dashboard updates

## 🚀 Quick Start

### Access Admin Panel

**Method 1: Direct File**
```
1. Open gift/admin.html in browser
2. Login with admin/admin123
3. Start managing!
```

**Method 2: Keyboard Shortcut**
```
1. Open gift/index.html
2. Press Ctrl + Shift + A
3. Enter password: admin123
4. Admin panel opens!
```

**Method 3: Secret Button**
```
1. Open gift/index.html
2. Wait 3 seconds
3. Click 🔒 button (bottom-left corner)
4. Enter password
```
