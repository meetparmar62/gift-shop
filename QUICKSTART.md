## ⚡ Quick Actions

### Add a New Product (30 seconds)
1. Click **"Products"** in sidebar
2. Click **"Add New Product"** button
3. Fill in:
   - Name: "Birthday Special Hamper"
   - Category: Select "Birthday"
   - Price: 1999
   - Image URL: Paste any Unsplash image URL
   - Badge: Select "New"
4. Click **"Save Product"** ✅

### Edit Existing Product
1. Go to Products section
2. Find product in table
3. Click **"Edit"** button
4. Make changes
5. Click **"Save Product"** ✅

### Delete a Product
1. Go to Products section
2. Find product in table
3. Click **"Delete"** button
4. Confirm deletion ✅

### Update Store Information
1. Click **"Contact Info"** in sidebar
2. Edit address, phone numbers, hours
3. Update Google Maps embed URL
4. Click **"Save Changes"** ✅

### Add Customer Review
1. Click **"Reviews"** in sidebar
2. Click **"Add New Review"**
3. Enter:
   - Customer name
   - Location
   - Rating (1-5 stars)
   - Review text
4. Click **"Save Review"** ✅

## 🔧 Quick Troubleshooting

### Can't Login?
- Check caps lock is OFF
- Verify username: `admin` (lowercase)
- Verify password: `admin123` (all lowercase)
- Clear browser cache if still having issues

### Data Not Saving?
- Check if browser allows localStorage
- Try in different browser
- Disable private/incognito mode
- Check browser console (F12)

### Website Not Updating?
- Hard refresh: Ctrl + Shift + R
- Clear browser cache
- Check dynamic-content.js is loaded
- Verify no JavaScript errors in console
